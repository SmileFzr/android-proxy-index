# Hiddify


## Overview

Hiddify is a cross-platform proxy client based on sing-box.


## Information

| Item | Value |
|-|-|
| Platform | Android |
| Kernel | sing-box |
| Framework | Flutter |
| Status | Active |


## Supported Protocols

- VLESS
- Reality
- Trojan
- Hysteria2
- TUIC
- Shadowsocks
