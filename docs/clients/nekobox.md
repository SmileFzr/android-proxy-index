# NekoBox


## Overview

NekoBox is an advanced Android proxy client supporting multiple kernels.


## Information

| Item | Value |
|-|-|
| Platform | Android |
| Kernel | sing-box / Xray |
| Language | Kotlin |
| Status | Active |


## Supported Protocols

- VLESS
- VMess
- Reality
- Trojan
- Hysteria2
- Shadowsocks
