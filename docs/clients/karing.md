# Karing


## Overview

Karing is a modern Android proxy client based on sing-box.

It focuses on:

- Modern UI
- Multi protocol support
- sing-box ecosystem
- Easy configuration


## Information

| Item | Value |
|-|-|
| Name | Karing |
| Platform | Android |
| Language | Dart |
| Framework | Flutter |
| Kernel | sing-box |
| Status | Active |


## Supported Protocols

- VLESS
- VMess
- Trojan
- Reality
- Hysteria2
- TUIC
- Shadowsocks
